# 

Репозиторий содержит шаблоны к сопроводительным документам для ВКР: заданию, отзывам руководителя и консультантов, титульному листу к раздаточному материалу.

Каждый из документов самостоятелен и может компилироваться отдельно. Файл `all.tex` объединяет все документы в один. Документы используют класс [отчета по ВКР](https://github.com/iktovr/diploma-latex-template), требования к компиляции аналогичные.

Информация о студенте, руководителе, консультантах и кафедре заполняется в файле `info.tex`, большинство полей перекочевало из отчета, но также есть новые. Остальное необходимо вписывать напрямую в файлы, такие места отмечены красным (`\textcolor{red}`).
